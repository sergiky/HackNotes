# Installation guide


1.

> sudo apt install snapd

2.

> wget https://dl.pstmn.io/download/latest/linux64 -O postman-linux-x64.tar.gz

3.

> sudo tar -xzf postman-linux-x64.tar.gz -C /opt

4. 

> sudo ln -s /opt/Postman/Postman /usr/bin/postman

5. 

> postman
