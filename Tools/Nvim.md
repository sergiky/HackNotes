# Config files



-----


``:edit $MYVIMRC`` --> Access to configuration file

To do a pattern and replace all coincident
````bash
:%s/,/\r/g
````

To indicate a substitution
> %s

Where is the comma ( , ) 
> /,/

Replace with a carriage return:
> \\r

To all the matches in the file
> /g
